 - - - Aplicativo Android Studio com Conexão a Banco de Dados via API Spring Boot - - -

Este é um projeto de exemplo que demonstra como criar um aplicativo Android Studio que se conecta
a um banco de dados através de uma API Spring Boot. O aplicativo é desenvolvido na plataforma
Android Studio e a API é construída usando Spring Boot.

- - - Objetivo - - -

O principal objetivo deste projeto é fornecer um exemplo de como criar um aplicativo Android que
possa se comunicar com um banco de dados através de uma API RESTful desenvolvida com Spring Boot.
Isso inclui operações básicas de CRUD (Create, Read, Update, Delete) para interagir com os dados no
banco de dados.

- - -Requisitos - - -

Para executar este projeto, você precisará das seguintes ferramentas instaladas em seu ambiente de
desenvolvimento:

Android Studio (versão 24 ou superior)
Java Development Kit (JDK)
Spring Boot (já incluso)
Banco de Dados (MySQL, PostgreSQL, etc.)

- - - Configuração do Projeto - - -

Clone este repositório em sua máquina local.
Abra o projeto do aplicativo no Android Studio.
Certifique-se de configurar a conexão com a API Spring Boot.
Execute a API Spring Boot em sua máquina local.
Configure o aplicativo Android para usar o endereço da API Spring Boot.
Execute o aplicativo no emulador Android ou em um dispositivo físico.

- - -Funcionalidades - - -

O aplicativo contém as seguintes funcionalidades:

Listagem de itens do banco de dados.
Adição de novos itens.
Atualização de itens existentes.
Exclusão de itens.
